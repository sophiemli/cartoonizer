What is Cartoonizer?:
The Cartoonizer program allows users to choose an image from their computer
and view a cartoonized version of that image. They also have the option to 
save the cartoonized version of the image.
 
Modules you will need to run Cartoonizer:
1. PIL- download link: http://www.pythonware.com/products/pil/
2. OpenCV- download link: http://sourceforge.net/projects/opencvlibrary/
3. numpy- download link: http://sourceforge.net/projects/numpy/

How to run Cartoonizer:
Open the file with IDLE and click Run Module under the Run menu.
